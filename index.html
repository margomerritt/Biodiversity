Belly Button Biodiversity Dashboard

Use the interactive charts below to explore the dataset

This interactive tool displays demographic info per test subject ID number. The demographic info includes the person’s ID number, ethnicity, gender, age, location, bellybutton type (BBTYPE) and washing frequency (WFREQ).

Three charts are created from this demographic info.

The first chart is a horizontal bar chart titled “Top 10 Bacteria Cultures Found”. This bar chart displays the top 10 bacterial species (OTUs) when an individual’s ID is selected from the dropdown menu on the webpage. The bar chart displays the sample_values as the values, the otu_ids as the labels, and the otu_labels as the hover text for the bars on the chart.

The second chart is a gauge chart titled “Belly Button Washing Frequency”. This chart displays the weekly washing frequency’s value and displays the value as a measure from 0-10 on the progress bar in the gauge chart.

The third chart is a bubble chart called “Bacteria Cultures Per Sample”. The otu_ids are the x-axis values, the sample_values are the y-axis values, and the otu_labels are the hover-text values.
